ResuMailer is a productivity Chrome Extension that helps job seekers send personalized resume emails in bulk via Gmail using Excel input. We take your privacy seriously and are committed to transparency about how your data is handled.

1. Data Collection & Storage

  • We do not collect or transmit any personal data to external servers.
  • All data including email, resume file path, and Excel content is stored locally in your browser using chrome.storage.local and the IndexedDB API.
  • Your Gmail app password is encrypted using AES encryption (via CryptoJS) and stored only on your local machine.
  • No data is synced, uploaded to the cloud, or accessed remotely.

2. Permissions Explanation

  • activeTab: To allow interaction only with the current tab, as needed.
  • scripting: Used to dynamically manage DOM actions in the extension UI.
  • storage: Used for saving extension preferences and local data inputs.
  • host permissions: Required to enable file handling via the File System Access API.

3. Security & Encryption

Sensitive fields like your Gmail password are encrypted using AES before storage. Your credentials never leave your system. The extension does not track, share, or monitor your usage.

4. No Analytics or Tracking

We do not use Google Analytics or any tracking mechanisms in this extension. Your usage remains entirely private.
